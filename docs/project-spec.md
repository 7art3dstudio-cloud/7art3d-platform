# 7Art3D Project Spec

## What it is
7Art3D is a multi-vendor 3D marketplace and creator ecosystem.

## Main roles
- Buyer
- Designer
- Seller
- Student

## MVP pages
- Homepage
- Explore
- Product page
- Projects feed
- Project detail
- Profile
- Student Hub
- Login / Signup
- Seller onboarding
- Upload product

## MVP goals
- clean marketplace
- designer portfolio system
- student learning hub
- secure and scalable foundation
